# F1 25 Super Modpack

Projeto pessoal para organizar, testar e documentar um modpack histórico para F1 25.

## Objetivo

Transformar o F1 25 em uma experiência mais completa e histórica, com foco em:

- liveries clássicas
- carros lendários
- sons de motores clássicos
- capacetes e macacões históricos
- performance balanceada
- estabilidade do jogo
- organização segura dos mods

## Filosofia do projeto

Este repositório não armazena arquivos de mods, executáveis, cracks ou conteúdo protegido.

O GitHub será usado apenas para:

- documentação
- links de fontes
- checklist de instalação
- registro de testes
- controle de alterações
- histórico do projeto

## Regras

1. Baixar mods apenas de fontes confiáveis.
2. Instalar um mod por vez.
3. Testar o jogo após cada instalação.
4. Registrar tudo no log.
5. Nunca subir arquivos de mods no GitHub.
6. Manter backup do jogo original.
7. Não usar mods online.
8. Evitar qualquer arquivo suspeito ou sem procedência.
