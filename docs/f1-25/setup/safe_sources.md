# Fontes seguras para mods

## Fontes principais

| Fonte | Uso | Status |
|---|---|---|
| OverTake / RaceDepartment | Fonte principal de mods | Aprovada |
| Nexus Mods | Fonte secundária, apenas com cautela | Parcial |
| GitHub | Ferramentas open-source, se aplicável | Parcial |

## Fontes evitadas

| Fonte | Motivo |
|---|---|
| Google Drive aleatório | Difícil validar procedência |
| MediaFire aleatório | Alto risco de arquivo reempacotado |
| Torrent | Alto risco e baixa rastreabilidade |
| YouTube sem página oficial | Fonte fraca para arquivos |
| Sites que pedem desativar antivírus | Rejeitado |
| Arquivos com crack, keygen ou activator | Rejeitado |

## Critérios para aceitar um mod

- página pública com descrição clara
- comentários ou histórico de uso
- autor identificável
- versão compatível com F1 25 atual
- instruções de instalação claras
- sem pedido para desativar antivírus
- sem executável suspeito
- sem crack, keygen, patcher obscuro ou ativador

## Critérios para rejeitar um mod

- arquivo enviado por link solto
- compactado com senha sem explicação
- instalador .exe obrigatório sem reputação
- pedido para executar como administrador sem motivo claro
- detecção forte no antivírus
- ausência total de documentação
