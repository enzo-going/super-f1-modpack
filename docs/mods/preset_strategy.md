# Estratégia de presets históricos

Este projeto não tem como objetivo apenas acumular mods aleatórios no F1 25.

A ideia é transformar o jogo em uma experiência histórica coerente, usando presets completos sempre que possível.

## Regra principal

Um mod só deve ser aprovado se tiver coerência histórica, visual, sonora e funcional.

Não faz sentido adicionar apenas a pintura de um carro histórico se o restante do conjunto continuar incoerente, como som moderno, grid atual, performance incompatível e pilotos errados.

## Prioridade de instalação

| Prioridade | Tipo de mod | Exemplo |
|---|---|---|
| Alta | Temporada completa | F1 2020 Season Mod |
| Alta | Pacote histórico completo | Mercedes W11 + pilotos + suits + helmets |
| Média | Complemento sonoro coerente | Som turbo-híbrido 2020 |
| Média | UI/logos/nomes coerentes | Equipes e pilotos de uma temporada específica |
| Baixa | Livery isolada | Apenas pintura clássica |
| Rejeitada | Mod sem contexto | Pintura aleatória ou fantasia |

## Presets planejados

### Preset 1 — F1 2020

Objetivo: recriar a temporada 2020 da forma mais completa possível.

Elementos desejados:

- grid completo de 2020
- liveries das equipes de 2020
- capacetes dos pilotos
- macacões das equipes
- logos e nomes coerentes
- performance aproximada da temporada
- som coerente com a era turbo-híbrida, se disponível

Carros/equipes principais:

- Mercedes W11
- Red Bull RB16
- Ferrari SF1000
- McLaren MCL35
- Renault R.S.20
- Racing Point RP20
- AlphaTauri AT01
- Alfa Romeo C39
- Haas VF-20
- Williams FW43

### Preset 2 — Era Schumacher/Ferrari

Objetivo: montar uma experiência baseada na era Ferrari de Michael Schumacher, preferencialmente com pacote completo.

Critérios mínimos:

- Ferrari histórica coerente
- capacete do piloto

### Preset 3 — Era Senna/McLaren

Objetivo: montar uma experiência baseada em Senna/McLaren, apenas se houver pacote suficientemente completo.

Critérios mínimos:

- McLaren clássica coerente
- capacete do piloto
- elementos visuais compatíveis
- som ou contexto histórico, se disponível

### Preset 4 — Era dominante recente

Objetivo: incluir carros modernos de grande peso histórico.

Possíveis focos:

- Mercedes W11 / Hamilton 2020
- Red Bull RB19 / Verstappen 2023

## Critérios de aprovação

Um mod só pode ser aprovado se:

1. vier de fonte confiável;
2. tiver compatibilidade clara com F1 25;
3. fizer parte de um preset coerente;
4. não exigir crack, keygen, ativador ou desativação de antivírus;
5. puder ser revertido via launcher, backup ou verificação da Steam;
6. for testado offline antes da instalação de outro mod;
7. não quebrar o conceito do preset.

## Critérios de rejeição

Um mod deve ser rejeitado se:

- for apenas estética solta sem propósito;
- misturar eras sem coerência;
- não tiver procedência clara;
- exigir executável suspeito;
- causar crash ou instabilidade;
- sobrescrever arquivos sem instrução clara;
- não tiver compatibilidade com F1 25.
