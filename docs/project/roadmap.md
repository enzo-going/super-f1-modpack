# Roadmap do projeto

## Objetivo geral

Construir um ecossistema organizado para jogar Fórmula 1 histórica e moderna com mods, usando:

- F1 25 como base moderna/oficial
- Assetto Corsa como base principal para carros e pistas históricas
- um futuro launcher próprio em Python para gerenciar perfis/presets

## Direção atual

O F1 25 foi validado com SERPs, mas não será forçado a virar um simulador histórico completo.

O Assetto Corsa será o foco principal do Super F1, por permitir carros completos, pistas antigas, física própria, sons próprios e maior controle de mods.

## Fases

1. Organizar documentação e estrutura do projeto
2. Configurar Assetto Corsa com Content Manager
3. Configurar CSP
4. Validar G923 no Assetto Corsa
5. Instalar primeiro carro F1 de alta qualidade
6. Instalar primeira pista
7. Criar presets históricos
8. Desenvolver mini launcher em Python
