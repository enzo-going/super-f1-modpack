# Regras do projeto

## Regra principal

O projeto não deve acumular mods aleatórios.

Cada mod deve fazer parte de uma experiência coerente, preferencialmente por preset histórico.

## F1 25

Uso planejado:

- jogo oficial moderno
- testes leves com SERPs
- mods apenas se forem coerentes e seguros

## Assetto Corsa

Uso planejado:

- foco principal do Super F1
- carros históricos completos
- pistas antigas
- física e sons próprios
- presets por era/temporada

## Segurança

Não usar:

- cracks
- keygens
- ativadores
- torrents suspeitos
- Google Drive aleatório
- executáveis sem origem confiável
- mods que pedem desativar antivírus

## GitHub

O repositório deve armazenar:

- documentação
- scripts
- configs
- logs
- estrutura do launcher

O repositório não deve armazenar:

- mods pagos
- arquivos grandes
- arquivos de jogo
- zips de mods
- executáveis baixados
