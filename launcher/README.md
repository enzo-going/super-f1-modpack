# Super F1 Launcher

Futuro launcher em Python para gerenciar perfis do projeto.

## Ideia

Permitir escolher modos/presets como:

- Assetto Corsa — F1 2020
- Assetto Corsa — Schumacher Era
- Assetto Corsa — Senna Era
- Assetto Corsa — Modern Dominance
- F1 25 — Default
- F1 25 — SERPs Base

## Escopo inicial

O launcher deve começar simples:

1. listar presets disponíveis
2. abrir Content Manager
3. abrir pastas de mods
4. mostrar status dos perfis
5. registrar logs básicos

## Fora do escopo inicial

- instalar mods automaticamente
- alterar arquivos críticos sem backup
- injetar arquivos com jogo aberto
- mexer em anticheat
